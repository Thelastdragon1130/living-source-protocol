# Authority Model

Authority in the Living Source Protocol flows in one direction:

Origin → Governance → Execution

## Source Rule
Authority begins with the initiating party.

## Governance Rule
Governance exists to preserve integrity and alignment.

## Execution Rule
Execution systems carry out instructions but do not create original authority.

---

## Why This Matters

This prevents platforms, institutions, or automated systems from behaving as though they are the author of human intent.
