# Living Source Coin (LSC)

## Purpose

Living Source Coin (LSC) is the native asset of the Living Source Protocol.

It functions as a coordination layer for governance, participation, and execution.

---

## Core Functions

- Governance participation
- Proposal voting
- Ecosystem coordination
- Access to protocol roles
- Value exchange within the system

---

## Positioning

LSC is not designed as a simple payment token.

It is a governance and utility asset tied to system interaction and participation.

---

## Design Principle

Value is derived from participation, contribution, and alignment with the protocol.
