# Tokenomics

## Overview

The tokenomics of LSC are designed to support governance, sustainability, and ecosystem growth.

---

## Allocation Categories

- Treasury Reserve
- Governance Incentives
- Ecosystem Development
- Community Participation
- Liquidity Provision
- Strategic Growth

---

## Design Principles

- No extraction without contribution
- Incentives aligned with participation
- Controlled emission tied to value creation
- Transparent allocation and tracking

---

## Value Model

LSC gains value through:

- Governance relevance
- Ecosystem usage
- Participation demand
- Role-based utility
