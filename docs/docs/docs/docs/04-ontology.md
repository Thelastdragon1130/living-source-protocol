# Ontology

The Living Source Protocol is built on three primary concepts:

## Living Source
The originating actor whose intent initiates action, instruction, and agreement.

## Central Node
The root reference point for alignment, truth, and governing integrity.

## Utilities
Systems that carry out execution, including institutions, AI, smart contracts, and platforms.

---

These three concepts establish the permanent hierarchy of the protocol:

Living Source → Central Node alignment → Utility execution
