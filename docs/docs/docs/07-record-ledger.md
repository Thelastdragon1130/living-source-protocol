# Record and Ledger Logic

The Living Source Protocol treats records as the foundation of enforceability.

## Record Chain

Intent → Expression → Record → Time Stamp → Verification → Ledger Memory

## Ledger Function

A ledger is a system of memory. It preserves what happened, when it happened, and who initiated it.

## Forms of Ledger

- Blockchain ledgers
- Legal records
- Administrative databases
- Archived communications

---

## Core Rule

If a record is not attributable, verifiable, and preserved, it is weak as evidence.
