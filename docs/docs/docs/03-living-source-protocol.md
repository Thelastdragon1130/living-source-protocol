# Living Source Protocol

## Overview

The Living Source Protocol defines the relationship between origin, governance, and execution.

It establishes a hierarchy where:

- The source initiates
- Governance aligns
- Utilities execute

---

## Core Principle

Authority originates from the source, not the system.

---

## Structure

### Origin Layer
The initiating authority responsible for intent and instruction.

### Governance Layer
The alignment layer ensuring consistency and integrity.

### Execution Layer
Systems that process and enforce instructions.

---

## Key Rule

Systems execute instructions. They do not define reality.
