# Architecture

The Living Source Protocol is structured into three primary layers.

## 1. Origin Layer
This is where intent begins. It includes authorship, consent, declaration, and instruction.

## 2. Governance Layer
This layer defines rules, alignment standards, permissions, and integrity requirements.

## 3. Execution Layer
This layer includes the systems that process, enforce, record, and automate instructions.

---

## Architectural Rule

Execution must always remain subordinate to origin and governance.

---

## System Outcome

This layered model ensures that:

- Authority remains with the source
- Governance preserves alignment
- Execution scales without overriding origin
